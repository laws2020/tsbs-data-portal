Authoritative Data

All datasets are produced according to national and international statistical standards. TSBoS is the apex statistical agency for Taraba State and the custodian of all official State statistics.

Regular Updates

Indicators are updated quarterly or annually depending on the series. Every release is version-controlled and archived permanently, with full provenance traceable through the tsbsR package.

Open Access

All published datasets are freely available. Researchers and developers can access data through this portal or programmatically via the tsbsR R package with a single function call.

Open Data Portal

Data in four
lines of R.

Install tsbsR and access any Taraba State dataset directly in R — no manual downloads, no spreadsheets.

1

Install the package

One command installs tsbsR and all dependencies from GitHub.

2

Authenticate once

Store your GitHub token securely. It persists across all R sessions.

3

Get any dataset

Call tsbs_get() with a category and optional period. Data arrives as a clean dataframe.

1 # Install tsbsR from GitHub
2 devtools::install_github("laws2020/tsbsR")
3 
4 library(tsbsR)
5 
6 # Authenticate once
7 tsbs_connect("ghp_your_token")
8 
9 # Get the latest prices dataset
10 df <- tsbs_get("prices")
11 
12 # Plot it instantly
13 tsbs_plot("prices", type = "line")
